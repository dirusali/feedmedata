Machine Learning Made Easy

Trainyourmodel is a free tool making easy to run machine learning algorithms without any coding skills.
In general, you can use Unsupervised Algorithms to classify unlabeled data and find hidden patterns.
On the other hand, Supervised Algorithms need a labeled training dataset as input. You can use them to make predictions.
This is an overview of the applications for the main machine learning algorithms:

- Unsupervised learning with categorical variables: association techniques like Apriori or Hidden Markov.

- Unsupervised learning with continuous data: clustering methods like K-Means of dimensionality reduction as in PCA.

- Supervised learning with categorical variables: classification algorithms like SVM, decision trees, naive-bayes or logistic regression.

- Supervised learning with continuous variables: linear o multivariant regression and trees/random forests

- Deep learning: you can use neural networks for both supervised and unsupervised data.
