How does Feedmedata work?

{% comment %} {% endcomment %}

Machine Learning Made Easy

Feedmedata is a free tool on which you can run machine learning algorithms without any coding skills.
You can use Unsupervised Algorithms to classify unlabeled data and find hidden patterns. On the other hand, Supervised Algorithms need a labeled training dataset as input. You can use them to make predictions. Please if your input data contains a target variable, make sure to place it in the last column of your csv file.
